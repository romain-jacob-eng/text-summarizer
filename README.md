# LLM Text Summarizer — Anthropic API

## Overview
A command-line text summarization tool powered by the Anthropic API (Claude) that generates structured summaries at three levels of detail from any input text file: a one-sentence summary, a concise paragraph, and a 5-bullet-point breakdown. Demonstrates real-world API integration with clean function separation across a modular Python structure.

## Technologies
Python, Anthropic Python SDK, python-dotenv

## How to run
python summarizer.py
